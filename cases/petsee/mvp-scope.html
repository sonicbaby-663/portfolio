Приоритизация MVP

Четыре релизные фазы. Внутри каждой — фичи по пяти тематическим блокам из HMW-раздела, привязанные к метрикам.

MVP — запуск

Минимальный набор для первой версии. Всё, что не решает проблему доверия или не ускоряет бронирование, — вне скоупа.

Доверие

  • Ролевой онбординг с выбором роли — первая точка формирования доверия к сервису
  • Верификация ситтера через Госуслуги — 78% владельцев ставят в топ-2 факторов выбора
  • Бейджи в карточке ситтера: «Верифицирован», «Топ-ситтер», «Прошёл обучение», «Мгновенное бронирование»
  • Дата обновления анкеты — косвенный сигнал активности, владельцы боятся «мёртвых» профилей
  • Просмотр всех отзывов с деталями (имя, тип питомца, дата)
  • Модуль обучения для ситтера — бейдж + приоритет в поиске
  • Поясняющие bottom sheets: «Что значит проверен», «Правила отмены», «Состав цены»

Путь к бронированию

  • Пошаговый поиск по модели Airbnb: услуга → дата → питомец → результаты
  • Расширенные фильтры и мультиселект по типу питомца, размеру, опыту ситтера
  • Карточка питомца в 4 блока с возможностью отложить заполнение
  • Мгновенное бронирование (Instant Book) — автоподтверждение без ожидания
  • Свайпы заявок для ситтера — принять/отклонить прямо из списка

Контроль во время передержки

  • Запрос отчёта — кнопка в чате + пуш ситтеру с напоминанием
  • Пуши ситтерам — 83% владельцев назвали фото/видео главным успокаивающим фактором

Коммуникация

  • Быстрые ответы в чате — 4 категории шаблонов
  • Цветовое выделение активного бронирования — закреплённый чат с плашкой «Активно»
  • Поддержка PetSee — FAQ + чат платформы + экстренный контакт

Монетизация

  • Реферальная программа — персональная ссылка + бонус за завершённое первое бронирование
  • Бейдж «Высокооплачиваемая услуга» + плейсхолдер рекомендуемой цены для ситтера

v1.1 — ~3 месяца после запуска

Фичи на основе первых данных. Цель — повысить конверсию в повторное бронирование и снизить время до принятия заявки.

Доверие

  • Видео-приветствие ситтера в анкете — живой ролик убедительнее трёх строк текста
  • История передержек с фотографиями — доказательство реальной работы для новых пользователей
  • Короткий вводный звонок или встреча до бронирования — превращает незнакомца в «знакомого ситтера»

Путь к бронированию

  • Рейтинг-фильтр ситтеров по числу успешных передержек
  • Пошаговый bottom sheet при завершении бронирования

Коммуникация

  • AI-контекстуализация быстрых ответов — подставляет имя питомца, даты, адрес из профиля
  • Приоритизация сообщений ситтеров на активных передержках

v2 — ~6 месяцев после запуска

Масштабирование и дифференциация. Фичи, требующие партнёрств или технической зрелости продукта.

Доверие

  • Видео-обзор жилья ситтера — короткий видеотур с модерацией
  • AI-рекомендации ситтеров на основе истории владельца

Путь к бронированию

  • Booking-календарь для ситтера — управление доступностью на даты вперёд

Монетизация

  • Динамическое ценообразование — рекомендации ситтерам на основе спроса
  • Программа лояльности — накопительные бонусы за бронирования

Backlog

Желанные фичи без ясной бизнес-модели или с высокой сложностью реализации. В текущем роадмапе не стоят.

Не в роадмапе

  • Чат с ветеринаром 24/7 — 66% хотят бесплатно, бизнес-модель нежизнеспособна без партнёрства
  • Страхование питомца — желанная функция для 64%, но не блокирует первое бронирование
  • Диагностика забытых ситтеров — напоминание владельцам при бездействии
  • Отдельное приложение для ситтера — при масштабировании
